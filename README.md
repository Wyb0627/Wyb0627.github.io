# Academic CV Website

This is a personal academic CV website built with HTML, CSS, and JavaScript. It automatically converts LaTeX CV content into a modern, responsive webpage.

## Features

- Modern, responsive design
- Automatic LaTeX to HTML conversion
- Clean and professional layout
- Mobile-friendly interface
- Easy to maintain and update

## Setup Instructions

1. Clone this repository to your local machine
2. Make sure all your LaTeX files are in the root directory
3. Update the content in your LaTeX files as needed
4. Test the website locally by opening `index.html` in your browser

## Deploying to GitHub Pages

1. Create a new repository on GitHub
2. Push this repository to GitHub:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/yourusername/your-repo-name.git
   git push -u origin main
   ```
3. Go to your repository settings on GitHub
4. Scroll down to the "GitHub Pages" section
5. Select the main branch as the source
6. Your website will be available at `https://yourusername.github.io/your-repo-name`

## File Structure

- `index.html` - Main webpage
- `styles.css` - Styling for the website
- `script.js` - JavaScript for content loading and interactivity
- `*.tex` files - Your LaTeX CV content
- `photo.jpg` - Your profile photo

## Customization

- Modify `styles.css` to change colors, fonts, and layout
- Update the LaTeX files to change content
- Replace `photo.jpg` with your own photo
- Edit `index.html` to modify the structure or add new sections

## Browser Support

The website is compatible with all modern browsers:
- Chrome
- Firefox
- Safari
- Edge
